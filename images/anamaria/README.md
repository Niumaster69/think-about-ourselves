## Recomendaciones técnicas

- Formato: `.jpg` (no `.jpeg`, no `.png`).
- Lado más largo mínimo: 1200 px.
- Peso recomendado por imagen: ≤ 500 KB.
- Nombres en minúsculas. **Exactos**.
